مقدمات و مفاهیم پایه طراحی واسط کاربر

user interface (ui) : 
فضایی است که تعامل میان انسان و ماشین در آن رخ می‌دهد.
رابط کاربری، بخش دیدنی و قابل‌تعامل یک ابزار است که کاربر مستقیماً با آن سروکار دارد هدف از این تعامل، اجازه دادن به اجرا و کنترل ماشین از سوی انسان است
خلاصه : هرچیزی که به زیبایی مربوط باشد مثلا طراحی یک دکمه زیبا است.

user experience (ux) : 
یک سبکی که کاربر بتواند به راحی با نرم افزار کارکند مثلا در پروژه بتوانید به راحتی خرید را انجام داد
خلاصه : هر چیزی که به تجربه رابط کاربری کمک کند مثلا این دکمه در کجا قرار بگیرد تا کلیک زیادی روی دکمه باشه

user interaction designer :
مسعول طراحی تعاملات بین کاربر و سیستم و تمرکز روی انیمیشن ها و نحوه انتقال صفحه ها و بازخوردهای سیستم و رفتار اجزا.

هدف : تجربه روان و قابل پیشبینی برای کاربر و طراحی یک محصول



طراحی یک محصول : 

1. Research and analysis of user needs  تحقیق و تحلیل نیاز کاربر 


2. Wireframe (چیدمان) and user flow (نقش کاربری) 


 Wireframe : اسکلت اویه بدون جزعیات اولی تمرکز روی چینش المان ها و مسیر کاربر 


3. ui design (طراحی بسری) 


4. interactive Prototype (ساخت پروت تایپ تعاملی) : 


5.Usability testing and optimization (تست کاربرد پذیری و بهینه سازی)
